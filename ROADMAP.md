## How to Request New Examples

Have an integration pattern you'd like to see? We welcome requests!

1. Check the [existing issues](https://github.com/Avela-Education/integration-cookbook/issues) to see if it's already requested
2. If not, [create a new issue](https://github.com/Avela-Education/integration-cookbook/issues/new?template=example_request.md)
3. Describe your use case and the integration pattern you need
4. Vote on existing requests by adding a thumbs up reaction
